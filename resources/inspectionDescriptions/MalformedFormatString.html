This inspection reports illegal syntax in format strings, format specifiers that is incompatible with the given arguments, insufficient arguments given the format string, etc.

The inspection works in the following places:

  • "…".format(…)
  • value.formatted("…")
  • String.format("…", …)
  • printf("…", …)
  • System.out.printf("…", …)
  • Interpolated strings

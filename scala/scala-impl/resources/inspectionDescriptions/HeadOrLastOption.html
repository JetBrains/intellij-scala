Reports when headOption or lastOption is emulated, suggests to actually use headOption or lastOption instead.

Example:


var seq: Seq[Int]
if (seq.nonEmpty) Some(seq.head) else None
seq.lift(0)

if (seq.size == 0) None else seq.lastOption
seq.lift(seq.size - 1)

After the quick-fix is applied:


var seq: Seq[Int]
seq.headOption
seq.headOption

seq.lastOption
seq.lastOption

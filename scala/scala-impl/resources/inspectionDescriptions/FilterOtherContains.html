Reports filter or filterNot on a Set and suggests replacing with intersect or diff if possible.

Example:


val set = Set(1, 2)
val others = Set(2, 3)
set.filter(others.contains(_))
set.filterNot(others.contains)
set.filter(x => !others.contains(x))
set.filterNot(x => !others.contains(x))

After the quick-fix is applied:


val set = Set(1, 2)
val others = Set(2, 3)
set.intersect(others)
set.diff(others)
set.diff(others)
set.intersect(others)

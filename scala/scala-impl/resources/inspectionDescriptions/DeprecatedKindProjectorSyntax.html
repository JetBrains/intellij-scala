Reports usage of `?` identifier in kind-projector syntax.

`?` identifier is going to be used for wildcard arguments in types in Scala 3 (see Wildcard Arguments in Types). It's usage in kind-projector type lambdas is deprecated and is going to be superseeded by `*`.

Example:


List[?]

After the quick-fix is applied:


List[*]

This inspection runs when Kind Projector plugin is enabled.

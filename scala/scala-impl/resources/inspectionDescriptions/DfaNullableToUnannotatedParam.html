This inspection identifies method calls where an argument that could potentially be null is passed to a parameter that that is not annotated with @Nullable, helping to detect potential NullPointerExceptions before they occur at runtime.

This inspection is only shown for targeted methods that are in the project's source code.

Example:


def test(@Nullable greeting: String): Unit = {
  sayHello(greeting)
}

def sayHello(greeting: String): Unit = {
  println(greeting.toLowerCase)
}

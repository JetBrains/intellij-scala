Reports if anonymous function can be converted to eta expansion with or without placeholder, also checks if placeholder can be removed from eta expansion.

Example:


def f(x: Any, y: Int): Unit = ()
val x = f(_, _)
val y: (Int, Int) => Unit = f _

After the quick-fix is applied:


def f(x: Any, y: Int): Unit = ()
val x = f _
val y: (Int, Int) => Unit = f

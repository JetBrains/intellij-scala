Reports comparing .length or .size and suggests replacing with .sizeIs.

Example:


var seq: Seq[Int]
seq.length == n
seq.length != n
seq.length < n
seq.length > n

seq.size == n
seq.size != n
seq.size < n
seq.size > n

After the quick-fix is applied:


var seq: Seq[Int]
seq.sizeIs == n
seq.sizeIs != n
seq.sizeIs < n
seq.sizeIs > n

seq.sizeIs == n
seq.sizeIs != n
seq.sizeIs < n
seq.sizeIs > n

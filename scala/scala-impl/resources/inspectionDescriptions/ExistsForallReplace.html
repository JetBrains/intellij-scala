Reports double negation in `exists` or `forall` calls and suggests removing it.

Example:


def condition(x: Int): Boolean = ???
!Seq(1, 2).exists(x => !condition(x))
!Seq(1, 2).forall(x => !condition(x))

After the quick-fix is applied:


def condition(x: Int): Boolean = ???
Seq(1, 2).forall(x => condition(x))
Seq(1, 2).exists(x => condition(x))
